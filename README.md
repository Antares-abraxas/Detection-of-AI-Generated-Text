# Detection of AI-Generated Russian Text via Syntactic and Punctuation Anomaly Analysis

## Project Overview
This research aims to develop a robust methodology for identifying AI-generated Russian texts. The core hypothesis focuses on the "Lexical Gap" — the structural and stylometric differences between human writing and Large Language Model (LLM) outputs. While modern LLMs excel at maintaining semantic coherence, they often exhibit specific patterns in syntax and punctuation that differ from natural human language.
