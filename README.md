# Detection of AI-Generated Russian Text via Syntactic and Punctuation Anomaly Analysis

## Project Overview
The rise of large language models has blurred the lines between human and synthetic output. This shift has birthed "textual deep-fakes". They pose a genuine threat to cybersecurity and academic integrity. The proposed method detects AI-generated texts by searching for syntactic and punctuation anomalies that are frequently met in LLMs’ outputs. Three key features are derived using the spaCy library: compression ratio of parts-of-speech, overall syntactical variance and punctuation entropy.
